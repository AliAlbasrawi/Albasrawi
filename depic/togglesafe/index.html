    togglesafe is a simple tweak that activates Safe Mode when the ringer/mute switch is turned on/off rapidly 3 times. Compatible with all jailbroken devices from iOS 7.x to iOS 11.1.2.


    No options to configure. Simply install and then use when needed to enter Safe Mode.
