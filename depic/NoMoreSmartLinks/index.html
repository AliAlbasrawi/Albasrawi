NoMoreSmartLinks

  • Remove Annoying Hyperlinks in iOS Text Messages!

    STABLE BETA REPORT ISSUES PLEASE

    NoMoreSmartLinks tweaks the stock iOS messages app by removing the "Smart Links" that are automatically created when someone texts you a time (9am), a date (January 5, 2017) and web links (www.reddit.com/r/jailbreak). Some users find this functionality bothersome and asked for this tweak on Reddit's r/jailbreak so I ported this tweak from a Flex patch made by the legendary Flex-Master himself: u/sinfool :)
