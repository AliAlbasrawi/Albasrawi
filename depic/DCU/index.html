DetailedCellularUsage

DEPICTION

  • Expose ALL cellular data usage including system apps & processes!

    BETA Software | iOS 8-10.x


    Have you ever wondered how much cell data is being used by system apps, processes or daemons? Uncloak all hidden cellular data usage details with DetailedCellularUsage! In addition to fully exposing cellular data usage, DetailedCellularUsage also unlocks the ability to restrict cellular data access to select system apps, processes & daemons*.


    DetailedCellularUsage is not enabled by default, so please visit Settings to enable the tweak after installation.


    *Only select system apps, processes and daemons will be made configurable. Please use CAUTION if you decide to restrict cellular access to system apps, processes or daemons!
